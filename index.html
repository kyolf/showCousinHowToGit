Steps in starting a new repository

  1. Open your terminal and change the path to the directory you want to start a repository
  2. Write the command 'git init' to initialize the repository
  3. Write the command 'git remote add origin repo-link' (Connect the local repo to github or bitbucket)
  4. Create files or make a change in file and save it
  5. Write 'git status' to check if you added files (it's red if you didn't)
  6. Write the command 'git add filename.ext' (to add each file singlehandlely) or 'git add .' (to add all files that were changed)
  7. Write 'git status' to check if you added files (it's green if you did)
  8. Write the command 'git commit -m "Message"' to save the changes to the local repository
  9. Write 'git status' to check if you commited the files (the file should not show)
  10. Write the command 'git push -u origin master' (-u tells git that when you do git push it will do git push origin master for you)
  11. Check if there are any code changes in your Github or Bitbucket

Steps in starting from an existing repository

  1. Open your terminal and change the path to the directory you want to start a repository
  2. Write the command 'git clone repo-link' (This copies all the files and folders in that repository and place it into the current folder)
  3. Repeat steps 4 - 11 from Steps in starting a repository anytime you create or make a change to a file

Steps in pushing into the repository (Github or Bitbucket)

  1. Repeat steps 4 - 11 from Steps in starting a repository anytime you create or make a change to a file

Steps in pulling from repository (Github or Bitbucket)

  1. See if your collaborator make any changes in your Github. If they did, do the below step.
  2. Write the command 'git pull origin master' or 'git pull' if you did 'git push -u origin master' before 'git pull'

Git Commands

  • git add . - Adds all the files that you changed into a list of files that you will save into the local repository
  • git add path-to-the-file/file.ext - Add that file to list of files that you will save into the local repository
  • git add remote origin repo-link - Connects your local repository to the online repository
  • git commit -m "Message" - Saves the files you changed into the local repository
  • git init - Creates a local repository
  • git pull origin master - Pulls all the changes that were made by a fellow collaborator and places those changes into the current local repository
  • git push -u origin master - -u sets origin master as the point where you will push to
    git push pushes everything commit you made in the local repository to the online repository
  • git status - Checks if you add or committed your files yet
