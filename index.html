Steps in starting a repository

  1. Open your terminal and change the path to the directory you want to start a repository
  2. Write the command 'git init' to initialize the repository
  3. Create files or make a change in file and save it
  4. Write 'git status' to check if you added files (it's red if you didn't)
  5. Write the command 'git add filename.ext' (to add each file singlehandlely) or 'git add .' (to add all files that were changed)
  6. Write 'git status' to check if you added files (it's green if you did)
  7. Write the command 'git commit -m "Message"' to save the changes to the local repository
  8. Write 'git status' to check if you commited the files (the file should not show)
  9. Write the command 'git remote add origin repo-link' (Connect the local repo to github or bitbucket)
  10. Write the command 'git push -u origin master' (-u tells git that when you do git push it will do git push origin master for you)
  11. Check if there are any code changes in your Github or Bitbucket
